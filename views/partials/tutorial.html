What is a CSV? A CSV document, or a comma separated values document, is similar to a spreadsheet but uses commas to separate columns of data.This format allows the program we use to easily take in data.

To get started, complete basic info about your essay.

Choose an upload format based on the content.

Select the amount of grading criteria you are grading on and enter them into the fields (Ex: Grammar, Voice, Ideas).

You can click any of the numbered boxes under the Text box to edit the text or grade of the essay you select, as well as delete an essay.

When you are completely done with add essays, you can download your CSV file by selecting "Download CSV."

You can change the scores and the document in the text box.

Once you are done, select the "Replace" button to save the changes to your grades and text.

If you choose not to change the document upon editing, you can click "Cancel" to cancel the editing.

If you wish to delete the selected essay, select the document number box and click the "Delete" button.
